# Useful Tools in Research and Data Analysis

这个仓库用于存放我在学习与实习过程中积累的各类数据分析工具。

## 核心内容
* **数据获取**: 包含针对特定平台的爬虫脚本。
* **数据清洗**: 基于 Pandas 的异常值处理与特征工程模版。
* **估值模型**: 估值项目相关的分析逻辑。
* **AI 行业研究**: 存放关于 AI 应用及模型深研的对比数据。

## 技术栈
* **语言**: Python, SQL
* **库**: Pandas, NumPy, Scikit-learn
* **环境管理**: Conda, Git, Obsidian

## 使用说明
所有的脚本均在 Python 3.x 环境下测试通过。建议通过以下方式快速启动：
1. 克隆仓库
2. 安装依赖 `pip install -r requirements.txt`
